# Unraid Templates

Docker templates for Unraid Community Applications.

## Applications

| Application | Description |
|---|---|
| [Stillwater](https://github.com/sydlexius/stillwater) | Artist metadata and image management for Emby, Jellyfin, and Kodi |

## Usage

Add this repository URL in Unraid under **Apps > Settings > Repository** to install these templates:

```
https://github.com/sydlexius/unraid-templates
```

## Support

Open an issue on the relevant application's GitHub repository.
